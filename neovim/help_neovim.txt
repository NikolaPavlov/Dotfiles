-------------------------------------------------------------------------------
* Moving
-------------------------------------------------------------------------------
    ) ---> move forward one sentence
    } ---> move forward one paragraph

    Ctrl+i ---> jump to your previous location
    Ctrl+o ---> jump back to where you were


    ---HML---
    H ---> move to the top of the screen
    M ---> move to the middle of the screen
    L ---> jump to the bott

    zz ---> move current line on the middle of the screen
    zt ---> move current line on the top of the screen

    Ctrl+U ---> move half screen up
    Ctrl+D ---> move half screen down

    m[x] ---> make a mark
    m[X] ---> make a mark in the current file (can go back from another file)
    '[x] ---> goto x
    '[X] ---> goto X (go to another file)

-------------------------------------------------------------------------------
* Spellchecking
-------------------------------------------------------------------------------
    :set spell spelllang=en_us
    ]s ---> go to the next misspelled word
    [s ---> go to last misspelled word
    z= ---> when on misspelled word get some suggestion

-------------------------------------------------------------------------------
* Search and find
-------------------------------------------------------------------------------
    f[char] ---> find char and land on it
    t[char] ---> find char and land before it
    /[str] ---> search for str
    % ---> go to match bracket
-------------------------------------------------------------------------------
* Marks
-------------------------------------------------------------------------------
:marks ---> show marks
'' ---> last place you where

-------------------------------------------------------------------------------
* Registers
-------------------------------------------------------------------------------

-------------------------------------------------------------------------------
* Macros
-------------------------------------------------------------------------------
    q[x] ---> start recording macro x
    q ---> stop recording
    @x ---> play back macro x

-------------------------------------------------------------------------------
* Splitting Windows
-------------------------------------------------------------------------------
    :hide ---> close current window

-------------------------------------------------------------------------------
* Folds
-------------------------------------------------------------------------------
    zR ---> open all folds
    zM ---> close all folds

-------------------------------------------------------------------------------
* Text objects
-------------------------------------------------------------------------------
    iw ---> inside word
    aw ---> around word

    is ---> inside sentence
    as ---> around sentence

    ip ---> inside a paragraph
    ap ---> around paragraph

    i' ---> inside '
    a' ---> around '

    i" ---> inside "
    a' ---> around '

    it ---> inside tag
    at ---> around tag

-------------------------------------------------------------------------------
Substitute
-------------------------------------------------------------------------------
    :%s/old/new/g ---> replace all occurances of old with new
    :s/old/new/g ---> replace all occurances of old with new in current line

-------------------------------------------------------------------------------
Buffers
-------------------------------------------------------------------------------
:buffers ---> display all buffers
:b [number] ---> go to [number] buffer
:bp ---> previous buffer
:bn ---> next buffer
:b# ---> go to where you been

-------------------------------------------------------------------------------
Formating
-------------------------------------------------------------------------------
    ,a= ---> select all text (,a) and formated it
    >> ---> indent currently marked text
    << ---> unindent currently marked text

-------------------------------------------------------------------------------
Surround Plugin
-------------------------------------------------------------------------------
    cs"' ---> change surrounding quotes from " to '
    cs'<q> ---> change ' to <q>
    ds" ---> delete " around something
    ysiw<em> ---> emphasize the current word (dot repeatable)

    Visual Mode: select anything, and then type S. You'll be brought to the
    bottom of the window. Now type in what you want to wrap that with, such as
    <div id='someid'> and then press enter.

-------------------------------------------------------------------------------
IDE like
-------------------------------------------------------------------------------
    <leader>g ---> jump to definition
    Ctrl+o ---> jump back
